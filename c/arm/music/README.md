## 简介

+ 支持 `GEC6818` 开发板的简易音乐播放器
+ 支持初始化音量

## 使用

### 编译

````bash
make
````

### 烧录

把当前目录下的所有  ***.mp3** 文件和 **bmp** 目录烧写到  **GEC6818** 开发板上。

### 运行

1. 默认音量运行

````bash
./app
````

2. 初始化音量运行

支持4种音量大小即 `-175, -15, 0, 10`

````bash
./app 0 # 按照-175的音量进行播放
./app 1 # 按照-15的音量进行播放
./app 2 # 按照默认音量进行播放
./app 3 # 按照10的音量进行播放
````

`madplay` 支持的音贝范围 `-175 to +18 dB`
